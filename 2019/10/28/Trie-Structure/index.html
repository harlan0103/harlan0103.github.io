Trie是一种特殊的Tree形式，用来存储String数据，每一个Trie node代表了a string。Trie结构多用于查找prefix。
一般的，我们使用创建一个TrieNode来表示Trie结构中的一个节点

class TrieNode {
    public static final size = 26;
    public TireNode[] children = new TrieNode[size];
    public boolean isWord;

}

// Usage: 
// Initialization: TrieNode root = new TrieNode();

因为我们使用了array来记录每一个TrieNode的子节点，那么访问每一个子节点就十分方便，但因为不是每一个节点都会包含所有26个字母的子节点，所以可能会浪费比较多的空间

public void insert(String word) {
    TrieNode current = root;

    // 对当前单词的每一个Char进行遍历
    for(int i = 0; i < word.length(); i++) {
        Character c = word.charAt(i);
        if(current.children[c - 'a'] == null) {
            // 如果当前c不在子节点中，我们更新子节点
            current.children[c - 'a'] = c;
        }
        current = current.children[c - 'a'];
    }
    // isWord表示当前节点是插入的word的截至
    current.isWord = true;
}
